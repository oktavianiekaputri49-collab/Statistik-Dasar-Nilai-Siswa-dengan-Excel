📊 Statistik Dasar Nilai Siswa dengan Excel

Deskripsi
Project ini berisi pengolahan data nilai siswa menggunakan Microsoft Excel untuk mempelajari statistik deskriptif. 
Data dianalisis untuk mendapatkan gambaran umum melalui perhitungan statistik dasar.

Dataset
Data yang digunakan meliputi:
- Nama lengkap siswa
- Nilai masing-masing siswa
Total data: 15 siswa

Analisis yang Dilakukan
Beberapa perhitungan statistik yang digunakan:
- Mean (rata-rata)
- Median
- Modus
- Nilai maksimum
- Nilai minimum
- Varians
- Standar deviasi

Hasil Analisis
- Rata-rata: 53.80  
- Median: 46  
- Modus: 28  
- Nilai tertinggi: 96  
- Nilai terendah: 16  
- Varians: 822.16  
- Standar deviasi: 28.67  

Tujuan
Project ini dibuat untuk:
- Memahami konsep statistik deskriptif  
- Melatih penggunaan rumus di Excel  
- Mengubah data mentah menjadi informasi yang lebih ringkas  

Pengembangan Selanjutnya
Project ini masih dapat dikembangkan dengan:
- Menambahkan visualisasi (grafik)  
- Menggunakan dataset yang lebih besar  
- Melakukan analisis lanjutan  

Penutup
Project ini merupakan bagian dari proses belajar saya di bidang statistika dan analisis data.
